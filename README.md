# K8blog
K8blog: A simple Green fresh fashion IT theme for hexo  <br>
(K8博客: 一款简洁、清爽、绿色、时尚、IT的Hexo主题)

#### Home: http://k8gege.org

![image](http://k8gege.org/k8img/k8blog/k8blog.png)

### 博客功能

- 3D博客标题
- 3D导航按钮
- 支持评论
- 文章缩略图
- 标签云
- 友情链接
- 文章搜索
- RSS订阅
- 图片懒加载
- URL持久化
- 文章分类
- 文章标签
- 站内搜索
- 访问统计
- 百度统计
